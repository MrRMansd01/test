🚀 سامانه مدیریت محتوای لندینگ (React + .NET Web API)یک پروژه فول‌استک (Full-Stack) سبک، سریع و بهینه برای مدیریت، جستجو و فیلتر داده‌ها همراه با قابلیت ماندگاری داده در فایل و خروجی مستقیم اکسل. این پروژه با ساختار تک‌صفحه‌ای (Landing Page) در فرانت‌اند و معماری مدرن Minimal API در بک‌آند دات‌نت پیاده‌سازی شده است.✨ ویژگی‌های کلیدی پروژه💾 ذخیره‌سازی ماندگار (Persisted Storage): ذخیره زنده، آنی و فیزیکی داده‌های ورودی فرم در فایل data.json سمت سرور دات‌نت بدون نیاز به تنظیم دیتابیس‌های سنگین.📊 خروجی اکسل هوشمند (Smart Excel Export): تولید و دانلود مستقیم فایل اکسل (.xlsx) بر اساس داده‌های فیلتر شده یا جستجو شده موجود در صفحه با استفاده از کتابخانه قدرتمند xlsx.🔍 فیلتر و جستجوی همزمان: اعمال فیلتر دسته‌بندی و سرچ متنی به صورت همزمان روی داده‌ها بدون تاخیر با بهره‌گیری از تکنیک Derived State در ریکت.🛠 مدیریت خطای شبکه: ساختار یافته با دستورات try/catch در فرانت‌اند و بک‌آند جهت پایداری بالا و هندل کردن پورت‌های ناهمگام.📱 طراحی مدرن و ریسپانسیو: استایل‌دهی تمیز، مینیمال و کاربرپسند با CSS مدرن و سازگاری کامل با تمامی نمایشگرها (موبایل، تبلت و دسکتاپ).📁 ساختار پوشه‌بندی پروژه (Architecture)test/
├── frontend/             # بخش فرانت‌اند (React + Vite)
│   ├── src/
│   │   ├── App.jsx       # لندینگ پیج، فرم، سرچ، فیلتر و منطق خروجی اکسل
│   │   ├── main.jsx      # نقطه ورود اصلی ریکت (Root Entry)
│   │   └── index.css     # استایل‌های عمومی و اختصاصی صفحه
│   ├── index.html
│   └── package.json
│
└── backend/             # بخش بک‌آند (.NET Minimal API)
    ├── Program.cs        # منطق اصلی سرور، تنظیمات CORS و مدیریت فایل JSON
    ├── data.json         # فایل فیزیکی ذخیره‌سازی داده‌ها (دیتابیس پروژه)
    └── backend.csproj    # فایل پیکربندی پروژه دات‌نت
🛠️ مراحل ساخت پروژه از صفر (Step-by-Step)مرحله ۱: ایجاد ساختار پوشه‌هایک پوشه اصلی به نام دلخواه (مثلاً test) ایجاد کرده و وارد آن شوید تا بخش‌های فرانت‌اند و بک‌آند به صورت کاملاً مستقل و تمیز در آن قرار بگیرند:mkdir test
cd test
مرحله ۲: ساخت و راه‌اندازی بک‌آند (.NET API)۱. یک پوشه مجزا برای سرور دات‌نت بسازید و وارد آن شوید:mkdir backend
cd backend
۲. با استفاده از CLI دات‌نت، یک پروژه وب‌آپی خام از نوع Minimal API ایجاد کنید (با پرچم --no-openapi جهت ساده‌سازی ساختار):dotnet new webapi -o . --no-openapi
۳. پکیج مدیریت و دسترسی CORS را به پروژه اضافه کنید تا به مرورگر و فرانت‌اند اجازه برقراری ارتباط بدون دردسر با سرور را بدهد:dotnet add package Microsoft.AspNetCore.Cors
۴. کدهای مربوط به خواندن و نوشتن روی فایل data.json، مدیریت درخواست‌های GET و POST و پیکربندی CORS را در فایل Program.cs پیاده‌سازی کنید.مرحله ۳: ساخت و راه‌اندازی فرانت‌اند (React)۱. به پوشه روت پروژه (test) برگردید و با استفاده از ابزار سریع Vite پروژه فرانت‌اند ریکت را پایه‌ریزی کنید:cd ..
npm create vite@latest frontend -- --template react
۲. وارد پوشه ساخته شده شوید:cd frontend
۳. پکیج‌های پیش‌فرض پروژه را نصب کنید:npm install
۴. پکیج کتابخانه xlsx را برای تبدیل ساختار JSON داده‌های جدول به فرمت خروجی اکسل نصب کنید:npm install xlsx
۵. فایل‌های اضافی در پوشه src را پاکسازی کرده و منطق اصلی لندینگ پیج، فرم‌های دوطرفه (Controlled Components)، توابع ارتباط با سرور (Fetch API) و توابع فیلترینگ را در فایل App.jsx پیاده‌سازی کرده و استایل‌های دلخواه را در index.css قرار دهید.⚙️ تکنولوژی‌های استفاده شدهبخش فرانت‌اند (Front-End):React (با استفاده از هوک‌های کاربردی useState و useEffect)Vite (ابزار ساخت و راه‌اندازی فوق‌سریع)JavaScript (ES6+)CSS3 (طراحی کاملاً واکنش‌گرا و مدرن)بخش بک‌آند (Back-End):.NET 10 (معماری بهینه Minimal APIs)C# (سی شارپ)کتابخانه‌های جانبی (Libraries):xlsx (تولید و دانلود فایل اکسل در سمت کلاینت)Microsoft.AspNetCore.Cors (پیکربندی دقیق قوانین CORS)
