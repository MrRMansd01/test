🚀 سامانه مدیریت محتوای لندینگ (Landing CMS)یک پروژه فول‌استک (Full-Stack) سبک، سریع و بهینه برای مدیریت، جستجو و فیلتر داده‌ها همراه با قابلیت ماندگاری داده در فایل و خروجی مستقیم اکسل. این پروژه با ساختار تک‌صفحه‌ای (Landing Page) در فرانت‌اند و معماری مدرن Minimal API در بک‌آند دات‌نت پیاده‌سازی شده است.✨ ویژگی‌های کلیدی پروژه💾 ذخیره‌سازی ماندگار (Persisted Storage): ذخیره زنده، آنی و فیزیکی داده‌های ورودی فرم در فایل data.json سمت سرور دات‌نت بدون نیاز به تنظیم دیتابیس‌های سنگین.📊 خروجی اکسل هوشمند (Smart Excel Export): تولید و دانلود مستقیم فایل اکسل (.xlsx) بر اساس داده‌های فیلتر شده یا جستجو شده موجود در صفحه با استفاده از کتابخانه قدرتمند xlsx.🔍 فیلتر و جستجوی همزمان: اعمال فیلتر دسته‌بندی و سرچ متنی به صورت همزمان روی داده‌ها بدون تاخیر با بهره‌گیری از تکنیک Derived State در ریکت.🛠 مدیریت خطای شبکه: ساختار یافته با دستورات try/catch در فرانت‌اند و بک‌آند جهت پایداری بالا و هندل کردن پورت‌های ناهمگام.📱 طراحی مدرن و ریسپانسیو: استایل‌دهی تمیز، مینیمال و کاربرپسند با CSS مدرن و سازگاری کامل با تمامی نمایشگرها (موبایل، تبلت و دسکتاپ).📁 ساختار پوشه‌بندی پروژه (Architecture)test/

    ├── data.json         # فایل فیزیکی ذخیره‌سازی داده‌ها (دیتابیس پروژه)
    └── backend.csproj    # فایل پیکربندی پروژه دات‌نت
🛠️ مراحل ساخت و راه‌اندازی پروژه (Step-by-Step)پیش‌نیازهامطمئن شوید که ابزارهای زیر روی سیستم شما نصب هستند:Node.js (نسخه ۱۸ یا بالاتر).NET SDK (نسخه ۸.۰ یا بالاتر)مرحله ۱: ایجاد ساختار پوشه‌هایک پوشه اصلی ایجاد کرده و وارد آن شوید:mkdir test
cd test
مرحله ۲: ساخت و راه‌اندازی بک‌آند (.NET API)۱. یک پوشه مجزا برای سرور دات‌نت بسازید و وارد آن شوید:mkdir backend
cd backend
۲. یک پروژه وب‌آپی خام از نوع Minimal API ایجاد کنید:dotnet new webapi -o . --no-openapi
۳. پکیج مدیریت و دسترسی CORS را به پروژه اضافه کنید:dotnet add package Microsoft.AspNetCore.Cors
۴. کدهای مربوط به خواندن و نوشتن روی فایل data.json و تنظیمات CORS را در فایل Program.cs پیاده‌سازی کنید.۵. برای اجرای سرور دات‌نت دستور زیر را در پوشه backend بزنید:dotnet run
مرحله ۳: ساخت و راه‌اندازی فرانت‌اند (React)۱. به پوشه روت پروژه (test) برگردید:cd ..
۲. با استفاده از ابزار Vite پروژه فرانت‌اند ریکت را پایه‌ریزی کنید:npm create vite@latest frontend -- --template react
۳. وارد پوشه ساخته شده شوید و پکیج‌های پیش‌فرض را نصب کنید:cd frontend
npm install
۴. پکیج کتابخانه xlsx را برای خروجی اکسل نصب کنید:npm install xlsx
۵. فایل‌های اضافی در پوشه src را پاکسازی کرده و منطق اصلی لندینگ پیج را در فایل App.jsx و استایل‌های دلخواه را در index.css قرار دهید.۶. برای اجرای فرانت‌اند دستور زیر را بزنید:npm run dev
⚙️ تکنولوژی‌های استفاده شده💻 بخش فرانت‌اند (Front-End)React (با استفاده از هوک‌های کاربردی useState و useEffect)Vite (ابزار ساخت و راه‌اندازی فوق‌سریع)JavaScript (ES6+)CSS3 (طراحی کاملاً واکنش‌گرا و مدرن)⚙️ بخش بک‌آند (Back-End).NET 8+ (معماری بهینه Minimal APIs)C# (سی شارپ)📚 کتابخانه‌های جانبی (Libraries)xlsx: تولید و دانلود فایل اکسل در سمت کلاینتMicrosoft.AspNetCore.Cors: پیکربندی دقیق قوانین CORS برای تعامل ایمن فرانت‌اند و بک‌آند
