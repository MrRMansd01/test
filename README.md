سامانه مدیریت محتوای لندینگ (React + .NET Web API)یک پروژه فول‌استک (Full-Stack) سبک و بهینه برای مدیریت، جستجو و فیلتر داده‌ها همراه با قابلیت ماندگاری داده در فایل و خروجی مستقیم اکسل. این پروژه با ساختار تک‌صفحه‌ای (Landing Page) در فرانت‌اند و معماری Minimal API در بک‌آند دات‌نت پیاده‌سازی شده است.✨ ویژگی‌های کلیدی پروژهذخیره‌سازی ماندگار (Persisted Storage): ذخیره زنده و فیزیکی داده‌های ورودی فرم در فایل data.json سمت سرور دات‌نت.خروجی اکسل هوشمند (Smart Excel Export): تولید فایل اکسل (.xlsx) بر اساس داده‌های فیلتر شده یا جستجو شده موجود در صفحه با استفاده از پکیج xlsx.فیلتر و جستجوی همزمان: اعمال فیلتر دسته‌بندی و سرچ متنی به صورت همزمان روی داده‌ها بدون تاخیر (تکنیک Derived State).مدیریت خطای شبکه: ساختار یافته با دستورات try/catch و مدیریت پورت‌های ناهمگام.طراحی مدرن و ریسپانسیو: استایل‌دهی تمیز با CSS مدرن و سازگار با تمامی نمایشگرها (موبایل و دسکتاپ).📁 ساختار پوشه‌بندی پروژه (Architecture)test/
├── frontend/             # بخش فرانت‌اند (React + Vite)
│   ├── src/
│   │   ├── App.jsx       # لندینگ پیج، فرم، سرچ، فیلتر و منطق اکسل
│   │   ├── main.jsx      # نقطه ورود ریکت
│   │   └── index.css     # استایل‌های عمومی و اختصاصی صفحه
│   ├── index.html
│   └── package.json
│
└── backend/             # بخش بک‌آند (.NET Minimal API)
    ├── Program.cs        # منطق اصلی سرور، تنظیمات CORS و مدیریت فایل JSON
    ├── data.json         # فایل فیزیکی دیتابیس پروژه
    └── backend.csproj
🛠️ مراحل ساخت پروژه از صفر (Step-by-Step Creation)مرحله ۱: ایجاد ساختار پوشه‌هایک پوشه اصلی به نام دلخواه (مثلاً test) ایجاد کرده و وارد آن شوید تا بخش‌های فرانت‌اند و بک‌آند به صورت کاملاً مستقل و تمیز در آن قرار بگیرند:mkdir test
cd test
مرحله ۲: ساخت و راه‌اندازی بک‌آند (.NET API)۱. یک پوشه مجزا برای سرور دات‌نت بسازید و وارد آن شوید:mkdir backend
cd backend
۲. با استفاده از دستور دات‌نت، یک پروژه وب‌آپی خام از نوع Minimal API ایجاد کنید (با پرچم --no-openapi جهت ساده‌سازی ساختار برای این تسک):dotnet new webapi -o . --no-openapi
۳. پکیج مدیریت و دسترسی CORS را به پروژه اضافه کنید تا به مرورگر و فرانت‌اند اجازه برقراری ارتباط با سرور را بدهد:dotnet add package Microsoft.AspNetCore.Cors
۴. کدهای مربوط به خواندن و نوشتن روی فایل data.json، مدیریت درخواست‌های GET و POST و پیکربندی CORS را در فایل Program.cs بازنویسی کنید.مرحله ۳: ساخت و راه‌اندازی فرانت‌اند (React)۱. به پوشه روت پروژه (test) برگردید و با استفاده از Vite پروژه فرانت‌اند ریکت را پایه ریزی کنید:cd ..
npm create vite@latest frontend -- --template react
۲. وارد پوشه ساخته شده شوید:cd frontend
۳. پکیج‌های پیش‌فرض پروژه را نصب کنید:npm install
۴. پکیج کتابخانه xlsx را برای تبدیل ساختار JSON داده‌های جدول به فرمت خروجی اکسل نصب کنید:npm install xlsx
۵. فایل‌های اضافی در پوشه src را پاکسازی کرده و منطق اصلی لندینگ پیج، فرم‌های دوطرفه (Controlled Components)، توابع ارتباط با سرور (Fetch API) و توابع فیلترینگ را در فایل App.jsx پیاده‌سازی و استایل‌های دلخواه را در index.css قرار دهید.⚙️ تکنولوژی‌های استفاده شدهFront-End: React (Hooks: useState, useEffect), Vite, JavaScript (ES6+), CSS3Back-End: .NET 10 (Minimal APIs), C#Libraries: xlsx (تولید فایل اکسل در فرانت‌اند), Microsoft.AspNetCore.Cors (مدیریت دسترسی فرانت به بک)
